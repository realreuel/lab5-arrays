🎯 Learning Objectives
  • ✓Define an array and explain why arrays are essential in structured programming
  • ✓Declare, initialise, and access indexed and associative arrays in PHP
  • ✓Traverse arrays using foreach with value-only and key => value syntax
  • ✓Modify arrays: add with array_push / [], remove with array_pop / unset
  • ✓Apply built-in functions: count, array_sum, sort, rsort, in_array, array_search, array_unique, array_reverse, array_slice, implode

  • ✓Perform a linear search and return the position of the found element
  • ✓Solve engineering analysis problems using arrays and loops (mean, max, min, above-average)
📌 Key rules before you start
🚫 Bubble Sort Rule In Task 3, you MUST implement bubble sort using nested for loops. Using PHP's sort(), rsort(), or any other built-in sort function is an automatic zero for that section AND triggers an academic integrity flag in the automated check.
⚠️ Pseudocode First For Tasks 3 and 4 you must write pseudocode in your PDF report before writing any PHP code. Marks are awarded for all three: pseudocode, flowchart, and working code.
📌 Array Indexing PHP arrays are zero-indexed. The first element is at index 0, not 1. This matters for bubble sort — your outer loop runs n-1 times, not n.
✅ foreach vs for Use foreach when you need every element. Use for with an index variable when you need to compare adjacent elements (bubble sort) or track position (linear search).

PHP Arrays Quick Reference
Declaring arrays
indexed & associative 
// Indexed
$scores = [72, 45, 88, 63];
echo $scores[0]; // 72

// Associative
$student = [
    "name" => "Jane",
    "gpa"  => 3.4
];
echo $student["name"];
foreach syntax
value-only & key=>value 
// Value only
foreach ($scores as $score) {
    echo $score . "<br>";
}

// Key => value
foreach ($student as $key => $val) {
    echo "$key: $val<br>";
}
Add & remove elements
push, pop, unset 
// Add to end
array_push($arr, "new");
$arr[] = "also new";

// Remove from end
$removed = array_pop($arr);

// Remove by index (gaps keys)
unset($arr[2]);

// Re-index after unset
$arr = array_values($arr);
Aggregate functions
count, sum, sort 
count($arr);       // number of elements
array_sum($arr);   // sum of all values
sort($arr);        // sort ascending (in-place)
rsort($arr);       // sort descending (in-place)

// Mean
$mean = array_sum($arr) / count($arr);
Search functions
in_array, array_search 
// Check existence (returns bool)
in_array(39.4, $temps);  // true/false

// Find index (returns index or false)
$idx = array_search(40.1, $temps);

// Always check with === not ==
if ($idx === false) {
    echo "Not found";
}
Utility functions
unique, reverse, slice, implode 
array_unique($arr);         // remove dupes
array_reverse($arr);        // reverse order
array_slice($arr, 2, 3);    // extract 3 from idx 2
implode(" | ", $arr);       // join as string
array_keys($arr);           // all keys
array_values($arr);         // re-index
Bubble sort pattern
manual — nested for loops 
$n = count($arr);
for ($i=0; $i < $n-1; $i++) {
  for ($j=0; $j < $n-1-$i; $j++) {
    if ($arr[$j] > $arr[$j+1]) {
      $t          = $arr[$j];
      $arr[$j]   = $arr[$j+1];
      $arr[$j+1] = $t;
    }
  }
}
Linear search pattern
manual — return index or -1 
function linearSearch($arr, $target) {
  $comparisons = 0;
  for ($i=0; $i < count($arr); $i++) {
    $comparisons++;
    if ($arr[$i] === $target) {
      return ["index" => $i,
              "comparisons" => $comparisons];
    }
  }
  return ["index" => -1,
          "comparisons" => $comparisons];
}
